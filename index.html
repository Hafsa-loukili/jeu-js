Documentation de la Calculatrice

Ce script JavaScript permet de créer une calculatrice simple avec des fonctionnalités de base telles que l'addition, la soustraction, la multiplication, la division et le calcul du pourcentage.

Étape 1 : Attente du chargement du document

Le code est placé à l'intérieur de l'événement DOMContentLoaded qui est déclenché lorsque le document HTML est entièrement chargé et prêt à être manipulé. 

    document.addEventListener('DOMContentLoaded', function () {
        // Le code ici sera exécuté une fois que le document HTML est entièrement chargé et prêt à être manipulé.
    });

Étape 2 : Récupération des éléments HTML

Nous récupérons deux éléments HTML importants :

  • display : Représente le champ de texte où les nombres et les opérations sont affichés.
  • buttons : Représente tous les boutons de la calculatrice.

    const display = document.getElementById('result');
    const buttons = document.querySelectorAll('.buttons button');

Étape 3 : Ajout des écouteurs d'événements pour les boutons

Nous ajoutons un écouteur d'événements "click" à chaque bouton de la calculatrice. Lorsqu'un bouton est cliqué, la fonction handleClick est appelée. 

    buttons.forEach(button => {
      button.addEventListener('click', handleClick);
    });

Étape 4 : Définition de la fonction handleClick

Cette fonction est exécutée chaque fois qu'un bouton de la calculatrice est cliqué. Elle récupère la valeur du bouton cliqué et effectue une action en fonction de cette valeur :

  • Efface l'affichage si le bouton "AC" est cliqué.
  • Supprime le dernier caractère de l'affichage si le bouton "DEL" est cliqué.
  • Évalue l'expression affichée si le bouton "=" est cliqué.
  • Ajoute la valeur du bouton à l'affichage pour les autres boutons.
En cas d'erreur lors de l'évaluation de l'expression, "Error" est affiché dans le champ de texte. 

    function handleClick() {
      const value = this.dataset.value;
      if (value === 'AC') {
        display.value = '';
      } else if (value === 'DEL') {
        display.value = display.value.slice(0, -1);
      } else if (value === '=') {
        try {
          display.value = eval(display.value);
        } catch (error) {
          display.value = 'Error';
        }
      } else {
        display.value += value;
      }
    }
